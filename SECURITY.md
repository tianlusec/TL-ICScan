# Security Policy

## Supported Versions

目前仅支持最新版本（Latest Release）的安全更新。

| Version | Supported          |
| ------- | ------------------ |
| > 0.5.x | :white_check_mark: |
| < 0.5.x | :x:                |

## Reporting a Vulnerability

如果您发现了本项目的安全漏洞，请**不要**在 GitHub Issue 中公开披露。

请通过以下方式联系维护团队：

- 发送邮件至：[security@example.com](mailto:security@example.com) (请替换为实际邮箱)
- 或通过 Keybase / PGP 加密联系（如有）。

我们会在 48 小时内确认收到报告，并定期同步修复进度。

### Scope

本项目的安全范围仅限于：
- `tianlu-intel-core` 的数据处理逻辑
- `tianlu_intel_collectors` 的采集逻辑
- 数据库存储的安全性

**注意**：本项目收集的漏洞情报（CVE 数据）本身包含公开的漏洞信息，这些信息不属于本项目的安全漏洞。
